# SuiProxy Android APK

Bản phát hành APK cho ứng dụng SuiProxy.

## Tải bản mới nhất

**[Tải SuiProxy APK (latest)](https://github.com/suiproxy/suiproxy-android-apk/releases/latest/download/SuiProxy.apk)**

Hoặc xem toàn bộ phiên bản tại trang [Releases](https://github.com/suiproxy/suiproxy-android-apk/releases).

## Cài đặt

1. Tải file `.apk` về máy Android
2. Cho phép cài đặt từ nguồn không xác định (Settings → Security → Unknown sources)
3. Mở file APK vừa tải và cài đặt
