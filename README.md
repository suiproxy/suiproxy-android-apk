# SuiProxy Android APK

Bản phát hành APK cho ứng dụng SuiProxy.

## Tải bản mới nhất

**[⬇ Tải SuiProxy APK](https://github.com/suiproxy/suiproxy-android-apk/releases/latest/download/SuiProxy.apk)**

Link trên là link cố định — luôn trỏ về bản mới nhất, không bao giờ đổi.

Phiên bản hiện tại: **v1.0.1** · [Xem tất cả releases](https://github.com/suiproxy/suiproxy-android-apk/releases)

## Cài đặt

1. Tải file `.apk` về máy Android
2. Cho phép cài đặt từ nguồn không xác định (Settings → Security → Unknown sources)
3. Mở file APK vừa tải và cài đặt
